# AeroGuard: AI-Powered Predictive Maintenance for Turbofan Engines

## Overview

AeroGuard is a machine learning project focused on predictive maintenance for aircraft turbofan engines. Using NASA's C-MAPSS engine degradation dataset, the system predicts the Remaining Useful Life (RUL) of an engine based on historical sensor measurements and operational settings.

The objective is to identify potential failures before they occur, enabling proactive maintenance planning, reducing operational costs, and improving reliability in aviation systems.

## Problem Statement

Traditional maintenance schedules often rely on fixed inspection intervals, which may lead to unnecessary maintenance or unexpected failures.

Predictive maintenance uses sensor data and machine learning models to estimate how much useful life remains in a component before maintenance becomes necessary.

This project predicts engine health by analyzing degradation patterns in NASA's turbofan engine simulation data.

## Technologies Used

## Dataset

NASA C-MAPSS Turbofan Engine Degradation Dataset

The dataset contains run-to-failure trajectories from multiple turbofan engines under varying operational conditions and fault modes.

## Results

The model predicts Remaining Useful Life (RUL) for each engine and generates maintenance recommendations based on predicted degradation levels.
